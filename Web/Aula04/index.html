O que é o HTML?

HTML é uma linguagem de marcação usada para estruturar páginas na web. Ele define títulos, parágrafos, imagens, links, etc.

O que é o CSS

CSS é uma linguagem usada para estilizar páginas HTML. Ele controla cores, fontes, espaçamentos e o layout dos elementos.
